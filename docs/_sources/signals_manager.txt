Data-driven filters (signals manager)
=====================================
.. image:: signals_manager.png
Данное  окно  служит  для  настройки  пространственных  режекторных  и  селекторных фильтров  с  помощью  применения
различных  методов. Также  есть  возможность отредактировать параметры временных фильтров. При нажатии на кнопку
Open для сигнала  из  соответствующей  строки  откроется  окно  для  соответствующего  столбцу анализа (SSD, CSP,
ICA). Каждый тип анализа производит разложение сырого сигнала на несколько компонент из которых можно построить
режекторный или селекторный пространственный фильтр. Ниже описаны окна для различных типов анализа
